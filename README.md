# 🎓 Results Box

A modern and responsive **Results Checker** web application built with **HTML, CSS, Bootstrap, and JavaScript**. Users can enter their exam score and instantly view the corresponding grade. The project features a clean UI, full-screen background image, responsive layout, and simple grade calculation logic.

## ⚙️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Enter your exam score.
5. Click **Check Data** to view your result.
